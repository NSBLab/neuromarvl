# NeuroMArVL Example data

Data which can be used with the [NeuroMArVL viewer](https://immersive.erc.monash.edu/neuromarvl/).

This repo contains all the necessary data used to generate the figures in the paper[^1], as well as the MATLAB code that was used to process/reformat the data.
